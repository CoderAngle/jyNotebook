## javascript语句

javascript语句向浏览器发出的命令。语句的作用是告诉浏览器该做什么

实例:javascript语句向id="demo"的HTML元素输出文本"hello world"

```
document.getElementById("demo").innerHTML="Hello World";
```

---

## javascript代码块

javascript语句通过代码块的形式进行组合

```
function myFunction()
{
    document.getElementById("demo").innerHTML=“Hello world”;
}
```
