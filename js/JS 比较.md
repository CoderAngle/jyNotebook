**比较和逻辑运算符用于测试 true 或 false。**

## 比较运算符

比较运算符在逻辑语句中使用，以测定变量或值是否相等。

给定 x=5，下面的表格解释了比较运算符：

| 运算符 | 描述 | 例子 |
| :--- | :--- | :--- |
| == | 等于 | x==8 为 false |
| === | 全等（值和类型） | x===5 为 true；x==="5" 为 false |
| != | 不等于 | x!=8 为 true |
| &gt; | 大于 | x&gt;8 为 false |
| &lt; | 小于 | x&lt;8 为 true |
| &gt;= | 大于或等于 | x&gt;=8 为 false |
| &lt;= | 小于或等于 | x&lt;=8 为 true |
