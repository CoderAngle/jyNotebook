## JavaScript 算术运算符

算术运算符用于执行变量与/或值之间的算术运算。

给定_y=5_，下面的表格解释了这些算术运算符：

| 运算符 | 描述 | 例子 | 结果 |
| :--- | :--- | :--- | :--- |
| + | 加 | x=y+2 | x=7 |
| - | 减 | x=y-2 | x=3 |
| \* | 乘 | x=y\*2 | x=10 |
| / | 除 | x=y/2 | x=2.5 |
| % | 求余数 \(保留整数\) | x=y%2 | x=1 |
| ++ | 累加 | x=++y | x=6 |
| -- | 递减 | x=--y | x=4 |

## JavaScript 赋值运算符

赋值运算符用于给 JavaScript 变量赋值。

给定_x=10_和_y=5_，下面的表格解释了赋值运算符：

| 运算符 | 例子 | 等价于 | 结果 |
| :--- | :--- | :--- | :--- |
| = | x=y |  | x=5 |
| += | x+=y | x=x+y | x=15 |
| -= | x-=y | x=x-y | x=5 |
| \*= | x\*=y | x=x\*y | x=50 |
| /= | x/=y | x=x/y | x=2 |
| %= | x%=y | x=x%y | x=0 |

_tips:数字+字符串=字符串_
