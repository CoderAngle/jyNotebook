## Server对象的常用属性

| 属性 | 说明 |
| :--- | :--- |
| ScriptTimeout | 获取和设置请求超时的时间\(以秒为单位\) |
| MachineName | 获取服务器的主机名称 |

---

## Server对象的常用方法

| 方法 | 说明 |
| :--- | :--- |
| CreateObject | 创建一个服务器的COM对象实例 |
| Execute | 使用另一页执行当前请求，执行完回到原页面 |
| MapPath | 将web服务器上的指定虚拟路径映射到实际的物理目录上 |
| Transfer | 停止当前页，在新的页面中执行请求 |
