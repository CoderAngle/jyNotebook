### 3.了解极验验证码

它是一个专注于提供验证安全的系统，主要验证方式是拖动滑动拼合图像。若图像完全拼合，则验证成功，即表单成功提交，否者重新验证

* 三角防护之防伪造

    def get_track(self,distanc):
        '''
        x=v0*t+0.5*a*t*t
        v=v0+a*t
        根据偏移量获取移动轨迹
        :param distanc: 偏移量
        :return: 移动轨迹
        '''
        # 移动轨迹
        track = []
        # 当前位移
        current = 0
        # 减速阈值
        mid = distanc * 4 / 5
        # 计算间隔
        t = random.randint(2,3)/10
        # 初速度
        v = 0

        while current < distanc:
            if current < mid:
                # 加速度为正2
                a = 2
            else:
                # 加速度为负3
                a = -3
            v0 = v
            # 当前速度v = v0+at
            v=v0+a*t
            # 移动距离x=v0*t+1/2*a*t*t
            move = v0*t+1/2*a*t*t
            # 当前位移
            current+=move
            # 加入轨迹
            track.append(round(move))
        return track

    def move_to_grap(self,slider,track):
