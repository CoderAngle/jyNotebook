### 语法

```
zip([iterable, ...])
```

### 作用

* zip\(\) 函数用于将可迭代的对象作为参数，将对象中对应的元素打包成一个个元组，然后返回由这些元组组成的对象，这样做的好处是节约了不少的内存
* 可以使用 list\(\) 转换来输出列表
* 如果各个迭代器的元素个数不一致，则返回列表长度与最短的对象相同，利用 \* 号操作符，可以将元组解压为列表

### 参数

* iterabl: 一个或多个迭代器

### 实例

```
>>> a = [1,2,3,4]
>>> b = ['a','b','c','d']
>>> zip(a,b)
<zip object at 0x000001BB7C96A2C8>
>>> list(zip(a,b))
[(1, 'a'), (2, 'b'), (3, 'c'), (4, 'd')]

>>> a1,b1 = zip(*zip(a,b)) # 与 zip 相反，zip(*) 可理解为解压，返回二维矩阵式
>>> a1
(1, 2, 3, 4)
>>> b1
('a', 'b', 'c', 'd')
```
