| 函数 | 描述 |
| :--- | :--- |
| acos\(x\) | 返回x的反余弦弧度值。 |
| asin\(x\) | 返回x的反正弦弧度值。 |
| atan\(x\) | 返回x的反正切弧度值。 |
| atan2\(y, x\) | 返回给定的 X 及 Y 坐标值的反正切值。 |
| cos\(x\) | 返回x的弧度的余弦值。 |
| hypot\(x, y\) | 返回欧几里德范数 sqrt\(x\*x + y\*y\)。 |
| sin\(x\) | 返回的x弧度的正弦值。 |
| tan\(x\) | 返回x弧度的正切值。 |
| 双曲函数 |  |
| acosh\(x\) | 返回x的反双曲余弦值 |
| asinh\(x\) | 返回x的反双曲正弦值 |
| atanh\(x\) | 返回x的反双曲正切 |
| cosh\(x\) | 返回x的双曲余弦值 |
| sinh\(x\) | 返回x的双曲正弦值 |
| tanh\(x\) | 返回x的双曲正切 |
