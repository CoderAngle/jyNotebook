## javascript 消息框

三种消息框:警告框、确认框、提示框

---

## 警告框

警告框经常用于确认用户可以得到某些信息

当警告框出现后，用户需要点击确认按钮才能继续进行操作

实例:

```
alert("文本");
```

## 确认框

确认框用于使用户可以验证或者接受某些信息

当确认框出现后，用户需要点击确定或者取消按钮才能继续进行操作

如果用户点击确认，那么返回值为true。如果用户点击取消，那么返回值为false

实例:

```
confirm("文本");
```

## 提示框

提示框经常用于提示用户在进入页面前输入某个值

实例:

```
prompt("文本","默认值")
```
