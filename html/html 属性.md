## HTML属性

* 属性以字典的形式出现，即:name="value"
* 属性在HTML元素的开始标签中规定

实例:

* ```HTML
  <a href="https://www.bilibili.com/">angle</a>
  ```

---

## 适用于大多数HTML元素的属性

| 属性 | 值 | 描述 |
| :---: | :---: | :---: |
| class | classname | 元素的类名 |
| id | id | 元素的唯一id |
| style | style\_definition | 元素的行内样式 |
| title | text | 元素的额外信息 |
