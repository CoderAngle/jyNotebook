# 如何使用样式表

### 有三种方式来插入样式表:

* 外部样式表
  ```HTML
  <head>
      <link rel="stylesheet" type="text/css" href="xx.css"></link>
  </head>
  ```
* 内部样式表
  ```HTML
  <head>
      <style>
          body{
              background-color:red;
          }
      </style>
  </head>
  ```
* 内联样式

```HTML
<p style="font-size:16px;">angle</p>
```
