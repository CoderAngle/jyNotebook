## html

全称“hyperTextmarkuplanguage”

### 基础标签

* 标签由一对`<>和</>`包括
  * 例如:
    * `<html>和</html>`
* 中文乱码问题
  * 在head标签中添加`<meta charset="utf-8"/>`
  * 在html中添加属性值,例如:`<html lang="en">`
    * lang告诉搜索引擎爬虫，网站是关于什么内容的
      * en:英文
      * zh:中文
      * ...
  * 编码集:
    * gb2312
    * gbk
    * Unicode
    * utf-8
    * ...
* title标签:标题
* meta标签
  * content:内容，name:关键字
* p标签:段落标签
* h标签
  * h1
  * h2
  * h3
  * h4
  * h5
  * h6
