算术运算符用于执行加、减、乘、除等数学运算

| 运算符 | 含义 | 示例 |
| :--- | :--- | :--- |
| + | 加法:两个操作数相加 | a+b=c |
| - | 减法:两个操作数相减 | a-b=c |
| \* | 乘法:两个操作数相乘 | a\*b=c |
| / | 除法:两个操作数相除 | a/b=c |
| % | 取模:返回两个操作数相除的余数 | a%b=c |
| \*\* | 幂:返回x的y次幂 | a\*\*b=c |
| // | 取整:返回除法时的整数部分 | a//b=c |
